# Chill 通用型程序设计语言 文档

Chill 语言是一门 同时支持动态与静态类型的、主要用于数据处理的高级程序设计语言。
Chill 的设计思想主要来源于，C语言 和 Lisp语言。
Chill 的主要实现 ICM 目前还在开发中。
